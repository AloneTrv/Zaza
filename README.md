- Download or clone this repo
- Run `install.bat` and and wait until it finishes installation
- Run `start.bat`

## How to run (others)
- Download or clone this repo
- Run `npm install` in console and and wait until it finishes installation
- Run `node index.js HWIDHERE` in console
- Ok thats it
